Moment 1 - Publicering

Jag valde Netlify som plattform att publicera min webbplats på. Netlify är en tjänst som är gratis och eftersom jag inte har använt den förut var den ett givet val. Det var väldigt lätt att komma igång, jag loggade in med hjälp av GitHub och sedan gjorde jag en ”drag and drop” med mappen jag ville ladda upp. Jag ändrade sitename för att skapa en enklare url.

För att underlätta arbetsflödet är det möjligt att skapa ett repository av mappen och lägga upp på GitHub som är kopplat till Netlify. I Netlify skapar man en ny site och då väljer man mellan de olika repositoryn som finns uppladdade på GitHub. Tack vare det här arbetssättet uppdateras webbplatsen direkt om man i Visual Studio Code gör ändringar och Commits till repositoryt.
