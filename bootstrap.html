Ramverket Bootstrap

Bootstrap är ett verktyg inom webbdesign för att enkelt skapa layout till webbplatser. Ramverket är gratis att använda och kan antingen laddas ner eller inkluderas i webbplatsen via CDN. Jag valde att inkludera via CDN. Ramverket erbjuder funktioner inom CSS, JavaScript samt SASS.

Startguide Bootstrap

Det finns många funktioner att använda sig av inom Bootstrap och i början kände jag mig väldigt ostrukturerad och visste inte alls hur jag skulle bygga upp webbsidan. Jag använde mig av olika guider och efteråt upptäckte jag att vissa hänvisade till äldre versioner av Bootstrap och alltså krävde andra länkar till stylesheeten. Det resulterade i väldigt ostrukturerad kod med blandning av Bootstrap5, 4 samt 3.

Fördelar

När man väl kommit in i funktionerna av ramverket så känns det enkelt att använda. Det är positivt att det även går att använda sig av JavaScript och SASS i webbplatserna. Att lägga till de olika funktionerna som krävdes för momentet såsom tabell och formulär med olika inmatningsalternativ var enkelt att göra. När jag väl hade hittat dokumentation till rätt version av Bootstrap.

Nackdelar

Eftersom det var det första ramverket som användes så hade jag väldigt svårt att komma igång och förstå mig på själva upplägget med ramverk. Mina förväntningar på vad det innebär att arbeta med ramverk var nog att det var väldigt mycket enklare än vad Bootstrap kändes som.
